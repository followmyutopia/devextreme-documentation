##### shortDescription
Specifies the scale's start value.

##### notUsedInTheme

---
Setting the start and end values is enough for the **RangeSelector** widget to create a scale and arrange ticks automatically. If required, you can also set [major](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/tickInterval/) and [minor](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/minorTickInterval/) tick intervals.
