<!--id-->dxRangeSelector.Options.scale.workdaysOnly<!--/id-->
===========================================================================
<!--default-->false<!--/default-->
<!--type-->Boolean<!--/type-->
===========================================================================

<!--shortDescription-->
Leaves only workdays on the scale: the [work week](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#workWeek) days plus [single workdays](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#singleWorkdays) minus [holidays](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#holidays). Applies only if the scale's [valueType](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#valueType) is *"datetime"*.
