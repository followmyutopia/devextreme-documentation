<!--shortDescription-->
Specifies options of the range selector's minor ticks.
<!--/shortDescription-->

<!--fullDescription-->
Major and minor ticks are calculated automatically based on the scale's [startValue](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#startValue) and [endValue](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#endValue) property values. Major ticks are made visible by default. You can make minor ticks visible as well, using their [visible](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/minorTick/#visible) property. In addition, you can set a custom minor tick interval or add custom minor ticks using the [tickInterval](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#tickInterval) and [customTickValues](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#customTickValues) properties respectively. You can also specify the length, width and color of the minor ticks using corresponding properties.
<!--/fullDescription-->
