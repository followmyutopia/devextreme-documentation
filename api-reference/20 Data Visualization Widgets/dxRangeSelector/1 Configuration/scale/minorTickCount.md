---
id: dxRangeSelector.Options.scale.minorTickCount
type: Number
default: undefined
---
---
##### shortDescription
Specifies the number of minor ticks between neighboring major ticks.

---
If you set both the **minorTickCount** property and the [minorTickInterval](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/minorTickInterval/) property, the **minorTickCount** property value will be ignored.
