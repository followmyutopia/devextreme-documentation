---
##### shortDescription
Specifies the length of aggregation intervals in pixels. May be ignored in favor of the [aggregationInterval](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/aggregationInterval/) option.

---
For data aggregation, the scale is divided into intervals. Series points that fall within the same interval get aggregated together. The **aggregationGroupWidth** option defines the length of each interval in pixels.
