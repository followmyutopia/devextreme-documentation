##### shortDescription
Specifies a fixed width for groups of bars with the same argument, measured in pixels. Takes precedence over the [barGroupPadding](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/chart/#barGroupPadding) option.

##### propertyOf
dxChartSeriesTypes.BarSeries,dxChartSeriesTypes.StackedBarSeries,dxChartSeriesTypes.FullStackedBarSeries,dxChartSeriesTypes.RangeBarSeries
