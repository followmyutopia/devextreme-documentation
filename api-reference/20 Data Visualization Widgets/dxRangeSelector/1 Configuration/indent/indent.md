<!--shortDescription-->
Range selector's indent options.
<!--/shortDescription-->

<!--fullDescription-->
The indent is used to precisely align scales of several widgets arranged vertically. This is the distance between the container edge and the range selector's scale (see the image below).

![RangeSelectorMargin ChartJS](/Content/images/doc/18_2/ChartJS/RangeSelectorIndent.png)

Note that if the indent is set very small, the [Scale Labels](/Documentation/Guide/Widgets/RangeSelector/Visual_Elements/#Scale/Scale_Labels) will be cut.
<!--/fullDescription-->
