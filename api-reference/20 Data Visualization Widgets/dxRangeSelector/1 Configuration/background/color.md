##### shortDescription
Specifies the background color for the **RangeSelector**.

---
You may need to use this property in the following cases:

* to set an appropriate background color when a specified background [image](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/background/image/) does not fill the entire background area;
* to set a custom background color for the [chart](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/chart/) displayed in the background.
