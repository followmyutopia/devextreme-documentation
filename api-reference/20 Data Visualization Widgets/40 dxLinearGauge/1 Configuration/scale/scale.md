Specifies the gauge's scale options.

<!--fullDescription-->
To divide a scale, set its <b>startValue</b> and <b>endValue</b> properties. The scale's major and minor ticks will be calculated automatically. Major ticks will be shown by default. To show minor ticks, set their <a href="/Documentation/17_1/ApiReference/Data_Visualization_Widgets/dxLinearGauge/Configuration/scale/minorTick">visible</a> property to <i>true</i>. You can set custom major and minor tick intervals, show custom major and minor ticks, and format tick labels. These and other scale options are available in the <b>scale</b> configuration object.
