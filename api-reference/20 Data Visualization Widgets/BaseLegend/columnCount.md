##### shortDescription
Arranges legend items into several columns.

---
Use this option when the legend is [oriented vertically]({basewidgetpath}/Configuration/legend/#orientation). Otherwise, use [rowCount]({basewidgetpath}/Configuration/legend/#rowCount).
