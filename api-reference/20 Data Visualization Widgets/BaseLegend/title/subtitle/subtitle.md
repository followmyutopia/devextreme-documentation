<!--id-->BaseLegend.title.subtitle<!--/id-->
===========================================================================
<!--type-->Object | String<!--/type-->
===========================================================================

<!--shortDescription-->
Configures the legend subtitle. The subtitle appears only if the [title]({basewidgetpath}/Configuration/legend/title/) is specified.
<!--/shortDescription-->

<!--fullDescription-->
To specify only the subtitle's text, assign it directly to this option. Otherwise, set this option to an object with the **text** and other fields specified.
<!--/fullDescription-->
