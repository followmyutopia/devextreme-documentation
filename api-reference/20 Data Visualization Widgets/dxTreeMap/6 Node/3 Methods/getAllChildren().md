---
id: dxTreeMapNode.getAllChildren()
---
---
##### shortDescription
Returns all nodes nested in the current node.

##### return: Array<dxTreeMapNode>
Nodes nested in the current node. Each member of this array has fields and methods described in the [Node](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Node) section.

---
This method allows you to obtain only direct descendants of the current node. To get all its descendants, call the [getAllNodes()](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Node/Methods/#getAllNodes) method.
