<!--id-->dxTreeMapNode.label(label)<!--/id-->

===================================================================

<!--shortDescription-->
Sets the label to the node.
<!--/shortDescription-->

<!--paramName1-->label<!--/paramName1-->
<!--paramType1-->String<!--/paramType1-->
<!--paramDescription1-->
A new label.
<!--/paramDescription1-->

<!--fullDescription-->
Usually, labels are provided by a [data source field](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Configuration/#labelField). To change the label of a specific node, call the **label** method passing the new text for the label as its parameter. 
<!--/fullDescription-->
