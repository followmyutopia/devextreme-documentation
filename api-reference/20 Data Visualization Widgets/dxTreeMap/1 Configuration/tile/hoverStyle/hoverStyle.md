<!--shortDescription-->
Specifies the appearance of tiles in the hover state.
<!--/shortDescription-->

<!--fullDescription-->
When the user pauses on a tile, this element responds by changing its style. Using the fields of the **hoverStyle** object, you can specify the [color](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Configuration/tile/hoverStyle/#color) of tiles and appearance of [tile borders](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Configuration/tile/hoverStyle/#border) in the hover state.

[note]If the widget visualizes a hierarchical structure, tiles are organized in groups. When the user pauses on a group, tiles belonging to that group also change their style to the one configured by the **hoverStyle** object.

#####See Also#####
- [hoverEnabled](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Configuration/#hoverEnabled) - enables/disables the hover feature.
<!--/fullDescription-->
