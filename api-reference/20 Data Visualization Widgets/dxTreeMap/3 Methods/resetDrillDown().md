<!--shortDescription-->
Resets the drill down level.
<!--/shortDescription-->

<!--fullDescription-->
Calling this method makes the root level the current one. If you need to drill one level up or down, use the [drillUp()](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Methods/#drillUp) method of the widget or the [drillDown()](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Node/Methods/#drillDown) method of a node.
