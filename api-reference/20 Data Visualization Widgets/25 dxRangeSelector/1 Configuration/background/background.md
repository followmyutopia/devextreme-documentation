<!--shortDescription-->
Specifies the options for the range selector's background.
<!--/shortDescription-->

<!--fullDescription-->
A background is an area that is painted in a specified color. You can change this color by setting the background's **color** property and by displaying an image specified by the [image](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/background/image/) configuration object. In addition, you can superimpose a chart on a background by specifying the [chart](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/chart/) configuration object. The chart will be shown with the specified background color. To show the chart without a background, set the background's **visible** property to **false**.

[note] You can use this option only when the range selector displays an [image](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/background/image/) or [chart](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/chart/). If not, use the [selectedRangeColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/#selectedRangeColor) and [tick](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/tick/) to customize the appearance of your widget.
