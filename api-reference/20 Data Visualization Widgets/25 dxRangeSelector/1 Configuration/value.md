<!--shortDescription-->
The selected range, initial or current.
<!--/shortDescription-->

<!--fullDescription-->
By default, the selected range is the entire [scale](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/). To specify a different range to be selected initially, assign an array of two values - start and end - to the **value** option. Depending on the [scale type](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#valueType), these values should be of Numeric, String or Date type.

When using [AngularJS](/Documentation/Guide/Getting_Started/Widget_Basics_-_AngularJS/Change_Options/)/[Knockout](/Documentation/Guide/Getting_Started/Widget_Basics_-_Knockout/Change_Options/), bind a scope property/observable value to the **value** option if you need to get or set this option at runtime. With [jQuery](/Documentation/Guide/Getting_Started/Widget_Basics_-_jQuery/Get_and_Set_Options/) only, use the [getValue()](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Methods/#getValue) and [setValue(value)](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Methods/#setValuevalue) methods for the same purpose.
