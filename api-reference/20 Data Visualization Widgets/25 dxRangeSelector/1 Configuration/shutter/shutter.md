Specifies range selector shutter options.

<!--fullDescription-->
Shutters are used in the **RangeSelector** widget with the background ([image](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/background/image/) or [chart](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/chart/)) to hide the non-selected range.
