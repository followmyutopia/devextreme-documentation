<!--/shortDescription-->

<!--fullDescription-->
<p>The following values are available.</p>

<ul>
<li><b>onlyPoint</b><br/>
Changes the appearance of the hovered point only.
</li>
<li><b>none</b><br/>
The appearance of the hovered series is not changed.
</li>
</ul>
