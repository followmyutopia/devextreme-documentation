<!--shortDescription-->
Gets a series within the chart's **series** collection by the specified name (see the [name](/Documentation/ApiReference/Data_Visualization_Widgets/dxPieChart/Configuration/series/#name) option).
<!--/shortDescription-->

<!--fullDescription-->
If a series with the specified name is not found, the method returns *null*.

To learn about series members you can use, refer to the description of the [Series](/Documentation/ApiReference/Data_Visualization_Widgets/dxPieChart/Chart_Elements/Series/) object.
