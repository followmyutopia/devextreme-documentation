<!--fullDescription-->
Use this property to set a predefined or custom palette. The colors listed in the specified palette will be used to draw chart series, their points, labels and tooltips. If the number of a series is greater than the number of colors in the palette, palette colors are repeated, but slightly modified.

You can override the color of individual series and their points, labels and tooltips by setting the **color** property of the corresponding object. In addition, you can define a custom palette to use in your charts. To learn how to do this, refer to the [Palettes](/Documentation/Guide/Data_Visualization/Common/Appearance_Customization/#Palettes) topic.
