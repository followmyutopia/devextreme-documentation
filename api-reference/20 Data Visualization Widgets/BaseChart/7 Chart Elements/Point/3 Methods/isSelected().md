<!--shortDescription-->
Provides information about the selection state of a point.
<!--/shortDescription-->

<!--returnType-->Boolean<!--/returnType-->
<!--returnDescription-->
Selection state of a point.
<!--/returnDescription-->

<!--fullDescription-->
When a point is selected, this method returns **true**. Otherwise, the returned value is **false**.

You can select a point using its **select()** method. To learn more about point and series selection, refer to the [Selection Handling](/Documentation/Guide/Widgets/Common/Data_Visualization_Widgets/Charts_-_End-User_Interaction/Selection_Handling) topic.
