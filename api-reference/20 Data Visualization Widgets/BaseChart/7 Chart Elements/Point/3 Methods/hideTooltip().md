<!--shortDescription-->
Hides the tooltip of the point.
<!--/shortDescription-->

<!--fullDescription-->
The **hideTooltip()** and **showTooltip()** methods allow you to change tooltip visibility in code. Call the point's **hideTooltip()** method to hide the tooltip of this point.
<!--/fullDescription-->
