id: MapLayerElement.applySettings(settings)
---
---
##### shortDescription
Applies the layer element settings and updates element appearance.

##### param(settings): Object
A [configuration object](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/) for the element.

---
The **settings** object should contain one or more options to be changed for the layer element. The options available to change using this method are listed in the [layers](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/) option description.

Note that you cannot change the [dataSource](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#dataSource) option for a separate layer element.
