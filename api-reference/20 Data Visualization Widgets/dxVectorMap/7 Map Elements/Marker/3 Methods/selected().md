<!--shortDescription-->
Provides information about the selection state of a marker.
<!--/shortDescription-->

<!--returnType-->Boolean<!--/returnType-->
<!--returnDescription-->
Selection state of the marker.
<!--/returnDescription-->

<!--fullDescription-->
When a marker is selected, this method returns **true**. Otherwise, the returned value is **false**. To select/deselect a marker at runtime, use the [selected(state)](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Map_Elements/Marker/Methods/#selectedstate) method.
<!--/fullDescription-->
