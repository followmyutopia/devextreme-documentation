---
id: dxVectorMap.Options.zoomFactor
type: Number
default: 1
---
---
##### shortDescription
Specifies a number that is used to zoom a map initially.

##### notUsedInTheme

---
Use this option to specify a zoom factor for a map while configuring it. This option accepts a value that is greater than 1. Note that the zooming is performed with relation to the [center](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/#center) of the map.
