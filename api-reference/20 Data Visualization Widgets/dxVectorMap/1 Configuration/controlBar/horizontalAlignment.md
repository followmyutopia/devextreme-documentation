<!--shortDescription-->
Specifies the position of the control bar.
<!--/shortDescription-->

<!--fullDescription-->
In addition to this option, use the [verticalAlignment](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/controlBar/#verticalAlignment) option to adjust the position of the control bar.

When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `HorizontalAlignment` enum. This enum accepts the following values: `Left`, `Center` and `Right`.
