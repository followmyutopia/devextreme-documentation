<!--shortDescription-->
Specifies the legend's position on the map.
<!--/shortDescription-->

<!--fullDescription-->
To position a legend on the map, use this option in conjunction with the [verticalAlignment](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/legends/#verticalAlignment) option.

When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `HorizontalAlignment` enum. This enum accepts the following values: `Left`, `Center` and `Right`.
<!--/fullDescription-->
