<!--shortDescription-->
Specifies the type of the legend grouping.
<!--/shortDescription-->

<!--fullDescription-->
The **VectorMap**'s legend supports three grouping types.

- **color**        
Can be used if you have grouped map markers or areas in several differently-colored groups. For more information, refer to the layers's [colorGroupingField](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#colorGroupingField) and [colorGroups](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#colorGroups) option descriptions.

- **size**        
Available for markers of a bubble [type](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#type) only. Can be used if you have grouped bubble markers in several groups and specified a size for bubbles in each group. For more information, refer to the [sizeGroupingField](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#sizeGroupingField) and [sizeGroups](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#sizeGroups) option descriptions.
