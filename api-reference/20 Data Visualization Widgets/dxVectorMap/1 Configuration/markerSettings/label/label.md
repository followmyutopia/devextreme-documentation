<!--shortDescription-->
Specifies marker label options.
<!--/shortDescription-->

<!--fullDescription-->
A marker label is a text that accompanies a [map marker](/Documentation/Guide/Widgets/VectorMap/Visual_Elements/#Markers). Usually, a marker label displays the name of a certain geographical point. To specify a text for a marker label, assign it to the [text](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/markers/#text) field of an object from the [markers](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/markers/) array when you [provide data for markers](/Documentation/Guide/Widgets/VectorMap/Providing_Data/#Data_for_Markers).
<!--/fullDescription-->
