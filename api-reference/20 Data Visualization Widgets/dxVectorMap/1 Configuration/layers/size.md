id: dxVectorMap.Options.layers.size

---
##### shortDescription
Specifies the size of markers. Setting this option makes sense only if the [layer type](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#type) is *"marker"* and the [elementType](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#elementType) is *"dot"*, *"pie"* or *"image"*.
