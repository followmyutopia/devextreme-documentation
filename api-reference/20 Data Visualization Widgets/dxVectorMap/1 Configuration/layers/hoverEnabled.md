<!--id-->dxVectorMap.Options.layers.hoverEnabled<!--/id-->
===========================================================================
<!--default-->true<!--/default-->
<!--type-->Boolean<!--/type-->
===========================================================================

<!--shortDescription-->
Specifies whether or not to change the appearance of a layer element when it is hovered over.
<!--/shortDescription-->

<!--fullDescription-->
If this option is set to **true**, you can specify colors for the hovered layer element and its border using the [hoveredColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#hoveredColor) and [hoveredBorderColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#hoveredBorderColor) options, respectively.
<!--/fullDescription-->
