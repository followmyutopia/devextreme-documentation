<!--shortDescription-->
Specifies the type of a marker element. Setting this option makes sense only if the [layer type](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#type) is *"marker"*.
<!--/shortDescription-->

<!--fullDescription-->
When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `VectorMapMarkerType` enum. This enum accepts the following values: `Dot`, `Bubble`, `Pie` and `Image`.
