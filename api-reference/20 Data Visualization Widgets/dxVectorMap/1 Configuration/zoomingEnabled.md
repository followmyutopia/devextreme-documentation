---
id: dxVectorMap.Options.zoomingEnabled
type: Boolean
default: true
---
---
##### shortDescription
Disables the [zooming](/concepts/05%20Widgets/VectorMap/40%20End-User%20Interaction/20%20Zooming/10%20Zooming.md '/Documentation/Guide/Widgets/VectorMap/End-User_Interaction/#Zooming') capability.

---
Setting this option to **false** disables a user's ability to zoom the map. However, you can still zoom the map in code using the [zoomFactor(zoomFactor)](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Methods/#zoomFactorzoomFactor) method.
