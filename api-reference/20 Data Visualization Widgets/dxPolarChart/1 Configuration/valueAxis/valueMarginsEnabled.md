---
id: dxPolarChart.Options.valueAxis.valueMarginsEnabled
type: Boolean
default: true
---
---
##### shortDescription
Indicates whether to display series with indents from axis boundaries.

---
Additionally, if you use a continuous axis, you can specify how far to extend the axis from its maximum and minimum values. Use the axis' [minValueMargin](/Documentation/ApiReference/Data_Visualization_Widgets/dxPolarChart/Configuration/valueAxis/#minValueMargin) and [maxValueMargin](/Documentation/ApiReference/Data_Visualization_Widgets/dxPolarChart/Configuration/valueAxis/#maxValueMargin) properties to do this.
