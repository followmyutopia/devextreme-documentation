<!--shortDescription-->
An object defining the configuration options for the [grid lines](/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Grid) of an [axis](/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Axes) in the **PolarChart** widget.
<!--/shortDescription-->

<!--fullDescription-->
Grid lines are the reference lines used to improve the readability of a chart's visual data. The **grid** object exposes the properties that allow you to specify visibility and appearance settings for axis grid lines. To learn more about axis grid lines and their options, refer to the [Grid](/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Grid) topic.
