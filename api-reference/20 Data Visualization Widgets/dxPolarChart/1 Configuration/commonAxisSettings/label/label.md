Axis labels represent textual values for [axis ticks](/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Axis_Ticks), which are not visible by default. Use the **label** object's properties to set common label options for all axes simultaneously.

To set a common option for labels of a particular axis, use the **label** object within the [argumentAxis](/Documentation/ApiReference/Data_Visualization_Widgets/dxPolarChart/Configuration/argumentAxis/) or [valueAxis](/Documentation/ApiReference/Data_Visualization_Widgets/dxPolarChart/Configuration/valueAxis/) configuration object.
