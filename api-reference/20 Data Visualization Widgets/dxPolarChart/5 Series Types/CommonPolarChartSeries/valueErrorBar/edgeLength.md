<!--shortDescription-->
Specifies the length of the lines that indicate error bar edges.
<!--/shortDescription-->

<!--fullDescription-->
A value in the range from 0 to 1 specifies the length relative to the [point diameter](/Documentation/ApiReference/Data_Visualization_Widgets/dxPolarChart/Configuration/commonSeriesSettings/point/#size) (for circular series points) or [angular width](/Documentation/ApiReference/Data_Visualization_Widgets/dxPolarChart/Configuration/commonSeriesSettings/#barWidth) (for bars).

Any other value specifies the length in pixels.
<!--/fullDescription-->
