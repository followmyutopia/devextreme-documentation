##### shortDescription
Occurs when one of the data source fields has an unsupported type.

---
Data source fields must contain values of numeric, string or date-time [types](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/#argumentType) only. Otherwise, the E2003 error occurs.
