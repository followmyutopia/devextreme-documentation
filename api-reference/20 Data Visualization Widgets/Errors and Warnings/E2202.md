===========================================================================

<!--shortDescription-->
Occurs when the [startValue](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#startValue) or [endValue](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#endValue) option of the **RangeSelector**'s scale is not valid.
<!--/shortDescription-->

<!--fullDescription-->
The **startValue** and **endValue** options accept a numeric or date-time value depending on the scale data type. If anything else is passed, the E2202 error occurs.

- [onIncidentOccurred](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/#onIncidentOccurred)
<!--/fullDescription-->
