<!--shortDescription-->
Occurs in the [RangeSelector](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/) widget when the selected range is invalid.
<!--/shortDescription-->

<!--fullDescription-->
Commonly, this error appears when either the [value](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/#value) option or the [setValue(value)](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Methods/#setValuevalue) method were used incorrectly. To troubleshoot, check that the range you are trying to select falls between the scale [start](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#startValue) and [end](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#endValue) values and that the type of the range values is the same as the [scale type](/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/scale/#valueType).
<!--/fullDescription-->
