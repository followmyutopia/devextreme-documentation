<!--shortDescription-->
Occurs if an invalid value was specified for a strip or constant line.
<!--/shortDescription-->

<!--fullDescription-->
A strip for an axis is specified by the **startValue** and **endValue** options of an object in the [strips](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/strips/) array. A constant line is specified by the **value** option of an object in the [constantLines](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/constantLines/) array. All these options accept numeric values. If you try to assign a value of another type to one of these options, the E2105 error occurs.
