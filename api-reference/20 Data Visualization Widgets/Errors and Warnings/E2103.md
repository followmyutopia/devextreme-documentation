<!--shortDescription-->
Occurs if an incorrect value was assigned to an option that accepts callback functions only.
<!--/shortDescription-->

<!--fullDescription-->
This error occurs when you try to assign anything other than a callback function to options like [customizeTooltip](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/tooltip/#customizeTooltip) in **Chart** or [onPointSelectionChanged](/Documentation/ApiReference/Data_Visualization_Widgets/dxPieChart/Configuration/#onPointSelectionChanged) in **PieChart**.
<!--/fullDescription-->
