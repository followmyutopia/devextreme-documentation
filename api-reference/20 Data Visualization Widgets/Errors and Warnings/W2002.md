<!--shortDescription-->
Appears when none of the data source objects contain a valid argument or value field.
<!--/shortDescription-->

<!--fullDescription-->
When providing data for **Chart** or a chart within **RangeSelector**, you must specify [argument](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/series/#argumentField) and [value](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/series/#valueField) fields for [chart series](/Documentation/Guide/Widgets/Chart/Series/Overview/). If the chart data source does not contain the specified fields, or none of their values are valid, the widget places the W2002 warning into the browser console.
