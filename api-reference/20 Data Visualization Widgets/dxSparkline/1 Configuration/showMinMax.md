---
id: dxSparkline.Options.showMinMax
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether or not to indicate both the minimum and maximum values on a sparkline.

---
When this option is set to **true**, the minimum and maximum values are highlighted and painted in the colors specified by the [minColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxSparkline/Configuration/#minColor) and [maxColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxSparkline/Configuration/#maxColor) options correspondingly.
