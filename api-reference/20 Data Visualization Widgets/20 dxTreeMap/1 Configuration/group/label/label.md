<!--shortDescription-->
Configures the group labels.
<!--/shortDescription-->

<!--fullDescription-->
Each tile or the header of a group contains a text label identifying what this tile or group represents. You can change the font of group labels or their visibility using the fields of the **group** | **label** object. The same properties of tile labels can be changed using the **tile** | [label](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Configuration/tile/label/) object.

To customize a specific label, use the [customize(options)](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Node/Methods/#customizeoptions) method of the node to which the label belongs.

[note]If the area occupied by a tile or a group header is too small to fit a label in it, the label will not be displayed.

#####See Also#####
- [labelField](/Documentation/ApiReference/Data_Visualization_Widgets/dxTreeMap/Configuration/#labelField) - specifies the data source field that provides texts for tile and group labels.
