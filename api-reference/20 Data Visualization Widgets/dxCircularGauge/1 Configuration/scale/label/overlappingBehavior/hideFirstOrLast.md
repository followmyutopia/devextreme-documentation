Specifies which label to hide in case of overlapping.

<!--fullDescription-->
When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `GaugeOverlappingBehavior` enum. This enum accepts the following values: `HideFirstLabel` and `HideLastLabel`.
