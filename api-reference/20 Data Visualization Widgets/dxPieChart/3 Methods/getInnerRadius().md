##### shortDescription
Gets the radius of the doughnut hole in pixels. Applies only when the [type](/Documentation/ApiReference/Data_Visualization_Widgets/dxPieChart/Configuration/#type) is *"doughnut"* or *"donut"*.

##### return: Number
The radius of the doughnut hole.
