##### shortDescription
Formats the point argument before it is displayed in the point label. To format the point value, use the [format](/Documentation/ApiReference/Data_Visualization_Widgets/dxPieChart/Configuration/series/label/#format) option.

---
[note]DevExtreme widgets provide a wide choice of [predefined formats](/Documentation/ApiReference/Common/Object_Structures/format/#type). If you are, however, going to use custom formats, link the <a href="https://github.com/jquery/globalize" target="_blank">Globalize</a> library to your project. Learn how to do this from topics in the [Installation](/Documentation/Guide/Getting_Started/Installation/Local_Scripts/) section.
