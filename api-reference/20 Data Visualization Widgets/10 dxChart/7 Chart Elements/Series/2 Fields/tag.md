<!--shortDescription-->
Returns the [tag](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/series/#tag) of the series.
<!--/shortDescription-->

<!--fullDescription-->
When defining a series, you can specify auxiliary information for it via the **tag** option. You can specify auxiliary information for each point as well. To do this, add a tag field to the data source and specify the name of the field via the series [tagField](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/#tagField) option.
<!--/fullDescription-->
