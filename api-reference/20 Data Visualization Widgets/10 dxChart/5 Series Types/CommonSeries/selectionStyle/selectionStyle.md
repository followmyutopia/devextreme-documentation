<!--d-->Common selection style settings<!--/d-->
===========================================================================
<!--type-->object<!--/type-->
===========================================================================

<!--shortDescription-->
Configures the appearance adopted by the series when a user selects it.
<!--/shortDescription-->

<!--fullDescription-->
[note]Though not provided out of the box, the selection capability can be implemented using the widget API. Refer to the [onPointClick](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#onPointClick) option description for details.

Declared in [commonSeriesSettings](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/), **selectionStyle** applies to all series in the chart. Declared in a [series](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/series/) configuration object, **selectionStyle** applies to this particular series only. The series-specific **selectionStyle** overrides the common one.
<!--/fullDescription-->
