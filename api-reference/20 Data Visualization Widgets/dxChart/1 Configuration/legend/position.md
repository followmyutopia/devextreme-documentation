<!--shortDescription-->
Specifies whether the legend is located outside or inside the chart's plot.
<!--/shortDescription-->

<!--fullDescription-->
In addition to this option, use the legend's [horizontalAlignment](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/legend/#horizontalAlignment), [verticalAlignment](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/legend/#verticalAlignment) and [orientation](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/legend/#orientation) options to specify the legend layout.
