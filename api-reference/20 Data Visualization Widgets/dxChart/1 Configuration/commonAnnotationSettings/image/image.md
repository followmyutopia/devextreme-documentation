<!--shortDescription-->
Configures the image to be displayed in the annotation. Applies only if the [type](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/annotations/#type) is *"image"*.
<!--/shortDescription-->

<!--fullDescription-->
To display an image, assign its URL to the **url** option. Use the **height** and **width** options to resize the image if needed. Otherwise, you can assign the URL directly to the **image** option.
<!--/fullDescription-->
