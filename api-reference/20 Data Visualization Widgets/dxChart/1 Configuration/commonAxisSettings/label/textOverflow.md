##### shortDescription
Specifies what to do with axis labels that overflow the allocated space after applying [wordWrap](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonAxisSettings/label/#wordWrap): hide, truncate them and display an ellipsis, or do nothing.
