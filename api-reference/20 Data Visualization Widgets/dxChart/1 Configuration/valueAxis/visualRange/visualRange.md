<!--shortDescription-->
Defines the axis's displayed range. Cannot be wider than the [wholeRange](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/wholeRange/).
<!--/shortDescription-->

<!--fullDescription-->

The specified visual range is adjusted automatically when the argument axis is zoomed and scrolled. Set the [adjustOnZoom](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#adjustOnZoom) to **false** to disable this.
