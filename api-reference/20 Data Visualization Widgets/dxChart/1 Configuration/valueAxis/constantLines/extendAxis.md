<!--shortDescription-->
Specifies whether to extend the axis's default [visual range](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/visualRange/) to display the constant line.
<!--/shortDescription-->

<!--fullDescription-->
The default visual range equals the data range. A constant line outside visual range is not displayed. Set this option to **true** to extend the range and display the constant line.
<!--/fullDescription-->
