##### shortDescription
Coupled with the **max** option, focuses the widget on a specific chart segment. Applies only to the axes of the *"continuous"* and *"logarithmic"* [type](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/#type).
