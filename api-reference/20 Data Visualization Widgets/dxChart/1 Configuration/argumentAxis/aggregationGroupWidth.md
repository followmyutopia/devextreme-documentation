<!--shortDescription-->
Specifies the length of aggregation intervals in pixels. May be ignored in favor of the [aggregationInterval](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/aggregationInterval/) option.
<!--/shortDescription-->

<!--fullDescription-->
For data aggregation, the argument axis is divided into intervals. Series points that fall within the same interval get aggregated together. The **aggregationGroupWidth** option defines the length of each interval in pixels.
