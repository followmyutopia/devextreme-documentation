<!--shortDescription-->
Specifies the key that enables panning when [dragToZoom](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/zoomAndPan/#dragToZoom) is **true**. Applies only on devices that use a mouse.
<!--/shortDescription-->

<!--fullDescription-->
[note] The Meta key depends on the keyboard: the Command key (&#8984;) on Macintosh keyboards; the Windows key (&#8862;) on Windows keyboards.
