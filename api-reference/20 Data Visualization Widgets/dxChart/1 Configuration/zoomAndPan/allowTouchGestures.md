---
id: dxChart.Options.zoomAndPan.allowTouchGestures
type: Boolean
default: true
---
---
##### shortDescription
Specifies whether users can use touch gestures to zoom or pan the chart. Applies only if zoom and pan are allowed for the [argument](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/zoomAndPan/#argumentAxis) or [value axis](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/zoomAndPan/#valueAxis).

---
Users can use the spread and pinch gestures to zoom in and out, and the vertical and horizontal drag gestures to pan the chart.
