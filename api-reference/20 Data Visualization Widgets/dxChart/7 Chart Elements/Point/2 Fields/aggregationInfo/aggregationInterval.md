##### shortDescription
Contains the length of the aggregation interval in axis units (numbers or dates). If the interval is set in pixels (using the [aggregationGroupWidth](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/#aggregationGroupWidth) option), it will be converted to axis units.
