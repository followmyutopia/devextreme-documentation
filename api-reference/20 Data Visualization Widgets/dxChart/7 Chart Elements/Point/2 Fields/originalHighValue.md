##### shortDescription
Contains the high value of the point. This field is useful for points belonging to a series of the [candle stick](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Series_Types/CandleStickSeries/) or [stock](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Series_Types/StockSeries/) type only.
