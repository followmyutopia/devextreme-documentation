##### shortDescription
Specifies the length of the lines that indicate error bar edges.

---
A value in the range from 0 to 1 specifies the length relative to the [point diameter](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/point/#size) (for circular series points) or [width](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/#barWidth) (for bars).

Any other value specifies the length in pixels.
