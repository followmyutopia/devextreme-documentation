<!--shortDescription-->
Substitutes the [standard point symbols](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/point/#symbol) with an image.
<!--/shortDescription-->

<!--fullDescription-->
To display an image instead of series points, assign its URL to the **url** option. If needed, resize the image using the **height** and **width** options. Otherwise, you can assign the URL directly to the **image** option.
