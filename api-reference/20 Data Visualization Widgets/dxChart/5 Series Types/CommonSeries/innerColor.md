##### shortDescription
Specifies a filling color for the body of a series point that visualizes a [non-reduced](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/reduction/) value. Applies only to [*candlestick* series](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Series_Types/CandleStickSeries/).

##### propertyOf
dxChartSeriesTypes.CandleStickSeries
