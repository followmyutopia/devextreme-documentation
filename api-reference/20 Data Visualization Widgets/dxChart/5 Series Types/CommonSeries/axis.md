<!--shortDescription-->
Binds the series to a value axis.
<!--/shortDescription-->

<!--fullDescription-->
When there are multiple value axes in a chart, series need to know exactly which axis they are bound to. By default, all of them are bound to the first axis in the [valueAxis](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/) array. To bind a series to another axis, assign the [name](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/#name) of the axis to the **axis** series property.
