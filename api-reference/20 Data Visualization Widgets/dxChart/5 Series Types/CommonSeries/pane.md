---
##### shortDescription
Specifies which [pane](/Documentation/Guide/Widgets/Chart/Panes/Overview/) the series should belong to. Accepts the [name](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/panes/#name) of the pane.

---
If this option is not specified, the series will belong to the [default pane](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#defaultPane).

#####See Also#####
- [panes](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/panes/) - declares a collection of panes.
