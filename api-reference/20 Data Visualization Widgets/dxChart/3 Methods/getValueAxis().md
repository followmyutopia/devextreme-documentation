##### shortDescription
Gets a value axis.

##### return: chartAxisObject
A value axis.
The first axis from the [valueAxis](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/) array in a [multi-axis chart](/Documentation/Guide/Widgets/Chart/Axes/Multi-Axis_Chart/), and the [defaultPane](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#defaultPane)'s value axis in a [multi-pane chart](/Documentation/Guide/Widgets/Chart/Panes/Multi-Pane_Chart/).
