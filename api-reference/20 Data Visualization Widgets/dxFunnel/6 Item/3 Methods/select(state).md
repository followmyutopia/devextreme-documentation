id: dxFunnelItem.select(state)
---
---
##### shortDescription
Selects or cancels the funnel item's selection.

##### param(state): Boolean
**true** selects the funnel item; **false** cancels the selection.

---
See [onItemClick](/Documentation/ApiReference/Data_Visualization_Widgets/dxFunnel/Configuration/#onItemClick) to learn how to implement selection.
