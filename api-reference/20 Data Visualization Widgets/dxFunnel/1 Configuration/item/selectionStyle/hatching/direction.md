<!--shortDescription-->
Specifies hatching line direction.
<!--/shortDescription-->

<!--fullDescription-->
The following values are available:

* **none**   
Does not apply hatching.
* **right**   
Draws diagonal hatching lines upwards from left to right.
* **left**   
Draws diagonal hatching lines downwards from left to right.

When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `HatchingDirection` enum. This enum accepts the following values: `Right`, `Left` and `None`.
<!--/fullDescription-->
