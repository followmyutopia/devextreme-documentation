---
##### shortDescription
Specifies the ratio between the height of the neck and that of the whole funnel. Accepts values from 0 to 1. Applies only if the [algorithm](/Documentation/ApiReference/Data_Visualization_Widgets/dxFunnel/Configuration/#algorithm) is *"dynamicHeight"*.
