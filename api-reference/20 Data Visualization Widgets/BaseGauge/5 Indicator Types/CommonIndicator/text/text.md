Specifies the appearance of the text displayed in an indicator of the *rangeBar* type.
<!--/shortDescription-->

<!--fullDescription-->
Note that the **indent** option of the **text** configuration object must not be set to 0 if you need to show the text.
<!--/fullDescription-->
