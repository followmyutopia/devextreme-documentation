<!--shortDescription-->
Configures the legend.
<!--/shortDescription-->

<!--fullDescription-->
The legend is a component that helps a user identify bars. The legend contains several items, one per bar, each consisting of a colored marker and text showing the bar's argument.

![DevExtreme BarGauge: Legend](/Content/images/doc/19_2/bargauge/visual-elements/legend.png)

You can make the legend visible by setting the **legend**.**visible** option to **true**.
<!--/fullDescription-->
