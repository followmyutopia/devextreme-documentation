##### shortDescription
Specifies the orientation of the range container. Applies only if the **geometry**.[orientation](/Documentation/ApiReference/Data_Visualization_Widgets/dxLinearGauge/Configuration/geometry/#orientation) option is *"horizontal"*.

---
This option specifies the range container orientation relative to an invisible scale line.
