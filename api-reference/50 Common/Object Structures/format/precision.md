##### shortDescription
Specifies a precision for values of a numeric format.

---
This option applies when the [type](/Documentation/ApiReference/Common/Object_Structures/format/#type) option has one of the following values.

- 'currency'
- 'fixedPoint'
- 'percent'
- 'decimal'
- 'exponential'
- 'largeNumber'
- 'thousands'
- 'millions'
- 'billions'
- 'trillions'
