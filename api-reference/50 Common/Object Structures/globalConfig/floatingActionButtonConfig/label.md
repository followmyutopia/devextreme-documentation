##### shortDescription
Specifies the text label displayed inside the FAB.

---
Use this option only if there are two or more [SpeedDialAction](/Documentation/ApiReference/UI_Widgets/dxSpeedDialAction/) components on a page. Otherwise, specify the [label](/Documentation/ApiReference/UI_Widgets/dxSpeedDialAction/Configuration/#label) option in the **SpeedDialAction**.
