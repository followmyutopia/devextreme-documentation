<!--shortDescription-->
The text's color in hexadecimal characters.
<!--/shortDescription-->

<!--fullDescription-->
This option accepts values in the following format: *"#RRGGBBAA"* where R (red), G (green), B (blue), and A (alpha) are hexadecimal characters (0-9, A-F). A is optional.
<!--/fullDescription-->
