id: dxEvent.isImmediatePropagationStopped()
---
---
##### shortDescription
Checks if the [stopImmediatePropagation()](/Documentation/ApiReference/Common/Object_Structures/dxEvent/Methods/#stopImmediatePropagation) method was called on this event object.

##### return: Boolean
**true** if **stopImmediatePropagation()** was called, **false** otherwise.
