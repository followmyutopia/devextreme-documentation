##### shortDescription
The color of the legend item's marker.

---
If the legend is [grouped by color](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/legends/source/#grouping), this option's value is different for each legend item. Otherwise, the value equals the [markerColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/legends/#markerColor).
