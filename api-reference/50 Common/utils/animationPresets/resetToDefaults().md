<!--id-->animationPresets.resetToDefaults()<!--/id-->
===================================================================
===================================================================

<!--shortDescription-->
Deletes all custom animations.
<!--/shortDescription-->

<!--fullDescription-->
Initially, the animation repository includes a set of [predefined animations](/Documentation/ApiReference/Common/Utils/animationPresets/). You can add custom animations as well. If you need to clear the repository from custom animations and leave only predefined ones, call the **resetToDefaults** method. It will [clear](/Documentation/ApiReference/Common/utils/animationPresets/#clearname) the repository and [register the predefined animations](/Documentation/ApiReference/Common/utils/animationPresets/#registerDefaultPresets) applying all these changes once.
<!--/fullDescription-->
