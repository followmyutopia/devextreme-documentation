<!--shortDescription-->
Deletes the specified animation or clears all the animation repository, if an animation name is not passed.
<!--/shortDescription-->

<!--paramName1-->name<!--/paramName1-->
<!--paramType1-->string<!--/paramType1-->
<!--paramDescription1-->
The name of the animation to delete from the animation repository.
<!--/paramDescription1-->

<!--fullDescription-->
[note]If you need to have [predefined animations](/Documentation/ApiReference/Common/Utils/animationPresets/) in the repository and delete only custom animations, call the [resetToDefaults()](/Documentation/ApiReference/Common/Utils/animationPresets/Methods/#resetToDefaults) method instead.
<!--/fullDescription-->
