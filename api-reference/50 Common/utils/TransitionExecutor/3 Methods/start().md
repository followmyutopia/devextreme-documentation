<!--id-->TransitionExecutor.start()<!--/id-->
===================================================================
===================================================================

<!--shortDescription-->
Starts all the animations registered using the [enter(elements, animation)](/Documentation/ApiReference/Common/Utils/TransitionExecutor/Methods/#enterelements_animation) and [leave(elements, animation)](/Documentation/ApiReference/Common/Utils/TransitionExecutor/Methods/#leaveelements_animation) methods beforehand.
<!--/shortDescription-->

<!--returnType-->Promise<void><!--/returnType-->
<!--returnDescription-->
A Promise that is resolved after all the animations are finished. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.
<!--/returnDescription-->
