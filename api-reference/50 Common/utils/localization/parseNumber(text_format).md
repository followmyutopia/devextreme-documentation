##### shortDescription
Parses a string into a numeric value.

##### return: Number
A numeric value equivalent to the specified string.

##### param(text): String
A string value to be parsed.

##### param(format): format
The [format](/Documentation/ApiReference/Common/Object_Structures/format/) of the string to be parsed.
