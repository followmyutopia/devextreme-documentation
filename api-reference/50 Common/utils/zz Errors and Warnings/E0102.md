##### shortDescription
Occurs when the **comparisonTarget** option is not specified for the ['compare'](/api-reference/10%20UI%20Widgets/dxValidator/8%20Validation%20Rules/CompareRule/CompareRule.md '/Documentation/ApiReference/UI_Widgets/dxValidator/Validation_Rules/CompareRule/') rule.

---
The value of the **comparisonTarget** option must be specified so that 'compare' rule can be checked.
