<!--shortDescription-->
Occurs when requesting a Url by the key that is not defined within the [EndpointSelector](/Documentation/ApiReference/Data_Layer/EndpointSelector/)'s configuration object.
<!--/shortDescription-->

<!--fullDescription-->
Check to make sure that the **EndpointSelector** key of the required URL is included into the configuration object.
