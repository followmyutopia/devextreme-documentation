<!--shortDescription-->
Specifies how widget items are aligned along the main direction.
<!--/shortDescription-->

<!--fullDescription-->
When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `BoxAlign` enum. This enum accepts the following values: `Start`, `End`, `Center`, `SpaceBetween` and `SpaceAround`.
<!--/fullDescription-->
