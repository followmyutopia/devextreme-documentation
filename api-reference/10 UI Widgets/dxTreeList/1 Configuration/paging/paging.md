Configures paging.
<!--/shortDescription-->

<!--fullDescription-->
Paging allows the widget to render rows by pages instead of rendering them at once. To enable paging, set the **paging**.[enabled]({basewidgetpath}/Configuration/paging/#enabled) option to **true**.
