---
id: EventsMixin.on(eventName, eventHandler)
---
---
##### shortDescription
Subscribes to an event.

##### return: {WidgetName}
The object for which this method is called.

##### param(eventName): String
The event's name.

##### param(eventHandler): function()
The event's handler.

---
Use this method to subscribe to one of the events listed in the [Events](/api-reference/10%20UI%20Widgets/EventsMixin/3%20Methods/on(events).md '{basewidgetpath}/Events/') section.
