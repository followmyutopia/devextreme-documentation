<!--shortDescription-->
Expands groups or master rows in a grid.
<!--/shortDescription-->

<!--paramName1-->groupIndex<!--/paramName1-->
<!--paramType1-->Number|undefined<!--/paramType1-->
<!--paramDescription1-->
The index of the groups to expand. Pass **undefined** to expand all groups. Pass *-1* to expand all master rows.
<!--/paramDescription1-->

<!--fullDescription-->
[note]This method cannot be called when using a remote [data source](/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/#dataSource).
