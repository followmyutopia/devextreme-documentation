<!--id-->dxDataGrid.collapseAll(groupIndex)<!--/id-->
===================================================================
===================================================================

<!--shortDescription-->
Collapses master rows or groups of a specific level.
<!--/shortDescription-->

<!--paramName1-->groupIndex<!--/paramName1-->
<!--paramType1-->Number|undefined<!--/paramType1-->
<!--paramDescription1-->
The group's level. Pass **undefined** to collapse all groups. Pass *-1* to collapse all master rows.
<!--/paramDescription1-->

<!--fullDescription-->
[note] The [rowCollapsed](/Documentation/ApiReference/UI_Widgets/dxDataGrid/Events/#rowCollapsed) event is not raised when you call this method.
