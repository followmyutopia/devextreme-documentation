<!--fullDescription-->
Sorting in the **DataGrid** widget can be carried out in a single or multiple mode. For a comprehensive overview of these modes, refer to the [Single Mode](/Documentation/Guide/Widgets/DataGrid/Sorting/#Sorting_in_UI/Single_Mode) or [Multiple Mode](/Documentation/Guide/Widgets/DataGrid/Sorting/#Sorting_in_UI/Multiple_Mode) articles.

If you must disable sorting, assign *"none"* to the **mode** option.

When configuring the widget using [ASP.NET MVC Wrappers](/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/), specify this option using the `GridSortingMode` enum. This enum accepts the following values: `Single`, `Multiple` and `None`.
