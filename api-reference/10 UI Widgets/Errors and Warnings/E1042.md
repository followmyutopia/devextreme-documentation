##### shortDescription
Occurs in the **DataGrid** or **TreeList** widget if the key field is not set.

---
If your data is a [JavaScript array](/concepts/05%20Widgets/DataGrid/05%20Data%20Binding/10%20Simple%20Array/10%20Array%20Only.md '/Documentation/Guide/Widgets/DataGrid/Data_Binding/Simple_Array/Array_Only/'), use the widget's [keyExpr](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/keyExpr.md '/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/#keyExpr') option to specify the key field.  Otherwise, use the **key** option in the store.
