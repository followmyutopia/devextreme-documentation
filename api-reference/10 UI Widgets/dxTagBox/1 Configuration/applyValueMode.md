<!--shortDescription-->
Specifies how the widget applies values.
<!--/shortDescription-->

<!--fullDescription-->
The option accepts the following values.

- **instantly**  
 Applies values when they are selected.

- **useButtons**  
 Applies values when a user clicks the "OK" button.
