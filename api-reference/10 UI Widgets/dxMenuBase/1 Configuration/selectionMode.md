<!--shortDescription-->
Specifies the selection mode supported by the menu.
<!--/shortDescription-->

<!--fullDescription-->
To select an item on click and to visually display the selection, set the [selectByClick](/Documentation/ApiReference/UI_Widgets/dxMenu/Configuration/#selectByClick) option to *true*. Note that in this case the **selectionMode** should be *'single'*.

When configuring the widget using [ASP.NET MVC Wrappers](/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/), specify this option using the `MenuSelectionMode` enum. This enum accepts the following values: `Single` and `None`.
