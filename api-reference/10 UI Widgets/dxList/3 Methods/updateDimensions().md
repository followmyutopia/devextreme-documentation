<!--id-->dxList.updateDimensions()<!--/id-->
===================================================================
===================================================================

<!--shortDescription-->
Updates the widget scrollbar according to widget content size.
<!--/shortDescription-->

<!--returnType-->Promise<void><!--/returnType-->
<!--returnDescription-->
A Promise that is resolved after the update process is completed. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.
<!--/returnDescription-->

<!--fullDescription-->
Use this method to update the position of list items after the widget element was resized.
