Specifies whether the widget enables an end-user to select only a single item or multiple items.
<!--/shortDescription-->

<!--fullDescription-->
When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `NavSelectionMode` enum. This enum accepts the following values: `Single` and `Multiple`.
