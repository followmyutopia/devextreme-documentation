<!--fullDescription-->
A grid column is considered hidden when its [visible]({basewidgetpath}/Configuration/columns/#visible) option is *false*. By default, the header of a hidden column is displayed in the column chooser. If you require this header to be omitted from the column chooser, set the **showInColumnChooser** option to *false*.
