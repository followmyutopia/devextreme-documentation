Specifies the grid edge to which the column is anchored.

<!--fullDescription-->
If the grid's [columnFixing]({basewidgetpath}/Configuration/columnFixing/) | **enabled** option is set to *true* and the column's [fixed]({basewidgetpath}/Configuration/columns/#fixed) option is set to *true*, you can specify to which grid column the column must be anchored. For this purpose, use the **fixedPosition** option.

When configuring the widget using [ASP.NET MVC Wrappers](/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/), specify this option using the `HorizontalEdge` enum. This enum accepts the following values: `Left` and `Right`.
