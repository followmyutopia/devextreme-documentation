---
##### shortDescription
Specifies whether users can enter a new cell value on a key press. Applies if **editing**.[mode]({basewidgetpath}/Configuration/editing/#mode) is *"cell"* or *"batch"*.

---
Users can only enter a new value when they press a key on the numeric or alphanumeric keypad.
