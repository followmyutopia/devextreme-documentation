- **Single**

- **Multiple**

When configuring the widget using [ASP.NET MVC Wrappers](/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/), specify this option using the `GridSortingMode` enum. This enum accepts the following values: `Single`, `Multiple` and `None`.
