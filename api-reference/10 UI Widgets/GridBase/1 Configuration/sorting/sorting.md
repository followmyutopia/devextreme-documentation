Specifies options of runtime sorting.
<!--/shortDescription-->

<!--fullDescription-->
In **DataGrid**, a user can sort records by columns at runtime. To make a column available for sorting, set its [allowSorting]({basewidgetpath}/Configuration/columns/#allowSorting) option to *true*. Sorting can be executed in various modes. To specify the required mode, use the **mode** property of the **sorting** configuration object.
