<!--shortDescription-->
Specifies the maximum height of the widget.
<!--/shortDescription-->

<!--fullDescription-->
This option accepts a value of one of the following types:

- **Number**  
The height in pixels.

- **String**  
A CSS-accepted measurement of height. For example, `"55px"`, `"80%"`, `"inherit"`.
<!--/fullDescription-->
