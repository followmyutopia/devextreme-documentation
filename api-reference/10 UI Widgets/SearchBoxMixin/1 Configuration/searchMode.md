<!--shortDescription-->
Specifies whether the widget finds entries that contain your search string or entries that only start with it.
<!--/shortDescription-->

<!--fullDescription-->
When using the widget as an [ASP.NET MVC Control](/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/), specify this option using the `CollectionSearchMode` enum. This enum accepts the following values: `Contains` and `StartsWith`.
<!--/fullDescription-->
