<!--shortDescription-->
Specifies the radio group layout.
<!--/shortDescription-->

<!--fullDescription-->
The following values are available.

- "vertical"  
    Radio buttons are arranged vertically - as a column.

- "horizontal"  
    Radio buttons are arranged horizontally - as a row.
