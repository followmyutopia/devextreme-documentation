---
id: PivotGridDataSource.Options.fields.allowSorting
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether a user can change the field's sorting.

---
When this option is set to **true**, clicking the field's header in the [field chooser](/Documentation/Guide/Widgets/PivotGrid/Visual_Elements/#Field_Chooser) and on the [field panel](/Documentation/Guide/Widgets/PivotGrid/Visual_Elements/#Field_Panel) changes the sorting direction. An arrow icon in the header indicates the current sorting direction.
