<!--shortDescription-->
Specifies whether to display the field's [totals](/Documentation/Guide/Widgets/PivotGrid/Visual_Elements/#Totals). 
<!--/shortDescription-->

<!--fullDescription-->
[note] Setting this option to **false** hides the field's totals and summary values - only the [grand totals](/Documentation/Guide/Widgets/PivotGrid/Visual_Elements/#Totals/Grand_Total_Row_and_Column) are displayed. Set the [showValues](/Documentation/ApiReference/Data_Layer/PivotGridDataSource/Configuration/fields/#showValues) option to **true** to display summary values.
