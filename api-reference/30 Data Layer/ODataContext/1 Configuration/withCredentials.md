<!--id-->ODataContext.Options.withCredentials<!--/id-->
===========================================================================
<!--default-->false<!--/default-->
<!--type-->Boolean<!--/type-->
===========================================================================

<!--shortDescription-->
Specifies whether to send cookies, authorization headers, and client certificates in a cross-origin request. 
<!--/shortDescription-->

<!--fullDescription-->
This option's value is passed to the underlying <a href="http://api.jquery.com/jQuery.ajax/#jqXHR" target="_blank">jqXHR object</a>.

Use the [beforeSend](/Documentation/ApiReference/Data_Layer/ODataContext/Configuration/#beforeSend) function to specify custom authorization headers.
<!--/fullDescription-->
