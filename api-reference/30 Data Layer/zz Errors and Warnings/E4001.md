##### shortDescription
Occurs when the an unknown aggregating function is specified.

---
This error may occur in widgets that include the "summary" feature. For example, if you specify a not-valid [summaryType](/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/summary/groupItems/#summaryType) for the **DataGrid** widget, this error will occur.
