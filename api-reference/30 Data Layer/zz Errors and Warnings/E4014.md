##### shortDescription
Occurs when an unknown key type is detected.

---
This error is specific for the [ODataStore](/Documentation/ApiReference/Data_Layer/ODataStore/) and [ODataContext](/Documentation/ApiReference/Data_Layer/ODataContext/) objects. The **keyType** property accepts the following values: String, Int32, Int64, and Guid.
