##### shortDescription
Occurs when an unknown filter operation is used.

---
The available operators are: "!", "=", "<>", ">", ">=", "<", "<=", "startswith", "endswith", "contains", "notcontains".

For more information on filtering syntax, see the [Filtering](/Documentation/Guide/Data_Layer/Data_Layer/#Reading_Data/Filtering) topic.
