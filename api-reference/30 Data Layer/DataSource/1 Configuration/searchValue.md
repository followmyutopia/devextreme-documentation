##### shortDescription
Specifies the value to which the search expression is compared.

---
You can use this option along with [searchExpr](/Documentation/ApiReference/Data_Layer/DataSource/Configuration/#searchExpr) and [searchOperation](/Documentation/ApiReference/Data_Layer/DataSource/Configuration/#searchOperation) to specify a simple filter. Use the [filter](/Documentation/ApiReference/Data_Layer/DataSource/Configuration/#filter) option for more complex filtering conditions. Filters are combined if you specify them in both ways.
