<!--type-->Array<dxCommand><!--/type-->
===========================================================================

<!--shortDescription-->
An array of [dxCommand](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxCommand/) components that are created based on the application's [navigation](/Documentation/ApiReference/SPA_Framework/HtmlApplication/Configuration/#navigation) option value.
<!--/shortDescription-->

<!--fullDescription-->
Use the array provided by this field to change the set of commands that are displayed by the application's global navigation control.
<!--/fullDescription-->
