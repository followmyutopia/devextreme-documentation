<!--shortDescription-->
Returns a [configuration object](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxView/Configuration/) used to create a [dxView](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxView/) component for a specified view.
<!--/shortDescription-->

<!--paramName1-->viewName<!--/paramName1-->
<!--paramType1-->String<!--/paramType1-->
<!--paramDescription1-->
The name of the requested view.
<!--/paramDescription1-->

<!--returnType-->Object<!--/returnType-->
<!--returnDescription-->
A configuration object of the dxView component representing the specified view.
<!--/returnDescription-->

<!--fullDescription-->
The name passed as the method's parameter should be the value of the [name](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxView/Configuration/#name) option specified for the requested **dxView** configuration object.
<!--/fullDescription-->
