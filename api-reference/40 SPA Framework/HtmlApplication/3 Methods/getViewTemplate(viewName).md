<!--shortDescription-->
Returns an HTML template of the specified view.
<!--/shortDescription-->

<!--paramName1-->viewName<!--/paramName1-->
<!--paramType1-->string<!--/paramType1-->
<!--paramDescription1-->
A name of the searched view template.
<!--/paramDescription1-->

<!--returnType-->jQuery<!--/returnType-->
<!--returnDescription-->
A jQuery object representing the searched view template.
<!--/returnDescription-->

<!--fullDescription-->
Use this method to get an HTML template of a view. The name passed as the method's parameter should be the value of the [name](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxView/Configuration/#name) option specified for the **dxView** component representing the searched view.
<!--/fullDescription-->
