<!--shortDescription-->
Occurs when an unexpected element is found within the [dxView](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxView/) element.
<!--/shortDescription-->

<!--fullDescription-->
The **dxView** element may include child elements either of the [dxCommand](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxCommand/) or [dxContent](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxContent/) type. For details on how to define views and fit them into the application's layouts, refer to the [Views and Layouts](/Documentation/Guide/SPA_Framework/Views_and_Layouts/) article.
<!--/fullDescription-->
