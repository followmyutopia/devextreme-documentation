The returned object contains the following fields:

- **isValid**: Boolean      
Indicates whether all the rules checked for the group are satisfied.

- **brokenRules**: Array    
Rules that failed to pass the check. Their structure is described in the [Validation Rules](/Documentation/ApiReference/UI_Widgets/dxValidator/Validation_Rules/) section.

- **validators**: Array     
**Validator** widgets included in the validated group.
