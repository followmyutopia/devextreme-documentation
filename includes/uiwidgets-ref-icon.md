This option accepts one of the following:

- The icon's URL
- The icon's name if the icon is from the [DevExtreme icon library](/Documentation/Guide/Themes/Icon_Library/)
- The icon's CSS class if the icon is from an external icon library (see [Use External Icon Libraries](/Documentation/Guide/Themes/Icon_Library/#Use_External_Icon_Libraries))
- The icon in the [Base64](https://en.wikipedia.org/wiki/Base64) format
