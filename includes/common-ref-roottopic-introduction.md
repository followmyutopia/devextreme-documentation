See the [Installation](/Documentation/Guide/Getting_Started/Installation/DevExtreme_Dependencies/) section (for JavaScript libraries) or the [Prerequisites and Installation](/Documentation/Guide/ASP.NET_MVC_Controls/Prerequisites_and_Installation/) section (for ASP.NET MVC framework) to find details on setting up DevExtreme with a particular library or framework.
