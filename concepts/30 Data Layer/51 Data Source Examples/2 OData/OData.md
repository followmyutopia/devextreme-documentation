[OData](http://www.odata.org) is a universal open protocol for consuming data APIs. The DevExtreme data layer provides a special [Store](/Documentation/Guide/Data_Layer/Data_Layer/#What_Are_Stores) implementation to access OData web services ([ODataStore](/Documentation/ApiReference/Data_Layer/ODataStore/)). 

Use ODataStore to access one OData entity specified by the URL, or the [ODataContext](/Documentation/ApiReference/Data_Layer/ODataContext/) object to communicate with an entire OData service.
