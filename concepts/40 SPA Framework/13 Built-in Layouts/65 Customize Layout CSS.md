Layout CSS includes classes from the [DevExtreme themes](/Documentation/Guide/Themes/Predefined_Themes/). You can customize the classes for any of the DevExtreme themes using the DevExtreme **Theme Builder**.

![DevExtreme Theme Builder](/Content/images/doc/17_2/Common/LayoutsInThemeBuilder.png)

For details, refer to the [Theme Builder](/Documentation/Guide/Themes/Theme_Builder/) article.
