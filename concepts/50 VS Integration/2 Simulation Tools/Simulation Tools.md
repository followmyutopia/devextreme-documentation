When developing mobile applications, you have to run them on different devices to see how they look, feel and function. The DevExtreme applications (those that are built using [DevExtreme project templates](/Documentation/Guide/VS_Integration/Project_Templates/) ) can be viewed on any device using the DevExtreme device-simulating tools.
