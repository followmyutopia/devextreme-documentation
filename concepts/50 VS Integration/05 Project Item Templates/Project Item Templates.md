The [views](/Documentation/Guide/SPA_Framework/Views_and_Layouts/) that are specific to [SPA Framework](/Documentation/Guide/Common/Introduction_to_DevExtreme/#Mobile_Development/Overview/SPA_Framework)-based applications can be added using one of the following DevExtreme project item templates. Use these project item templates in the [Basic Application](/Documentation/Guide/VS_Integration/Project_Templates/#Basic_Application) and [Multi-Channel Application](/Documentation/Guide/VS_Integration/Project_Templates/#Multi-Channel_Application) projects - the projects where the DevExtreme SPA Framework is used to build the application.
