The [FileUploader](/Documentation/ApiReference/UI_Widgets/dxFileUploader/) widget enables an end user to specify a file or several files to be uploaded to the server, and upload them synchronously or asynchronously. The widget is based on an **input** element with the **type** attribute set to *"file"*.
