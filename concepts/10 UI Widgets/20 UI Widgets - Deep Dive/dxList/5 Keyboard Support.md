An end-user can use the following keys to interact with the widget.

- Up/down arrow  
 Highlight the previous/next item.

- Page up/page down  
 Highlights the item located a page above/below the current one.

- Home/End  
 Highlight the first/last element.

- Enter (Space)  
 Select the current element.

- Delete  
 Remove the current item. Available if the [allowItemDeleting](/Documentation/ApiReference/UI_Widgets/dxList/Configuration/#allowItemDeleting) option is set to *true*.

- Shift + up/down arrow  
 Transpose the current and previous/next items. Available if the [allowItemReordering](/Documentation/ApiReference/UI_Widgets/dxList/Configuration/#allowItemReordering) option is set to *true*.
