You can specify which views are available within the switcher. For this purpose, use the widget's [views](/Documentation/ApiReference/UI_Widgets/dxScheduler/Configuration/#views) option.

    <!--JavaScript-->
    var schedulerOptions = {
        // ...
        views: ['week', 'month']
    }

When the Scheduler is displayed on a mobile device, it uses a drop-down menu to switch views. To specify whether or not the widget uses drop-down menu instead of tabs, use the [useDropDownViewSwitcher](/Documentation/ApiReference/UI_Widgets/dxScheduler/Configuration/#useDropDownViewSwitcher) option.

    <!--JavaScript-->
    var schedulerOptions = {
        // ...
        useDropDownViewSwitcher: true
    }
