Resource headers display information about resources, assigned to [appointments](/Documentation/Guide/UI_Widgets/Scheduler/Resources/#Assign_Appointments_to_Resources). They are visible only when the appointments are grouped by resources. 

![Scheduler View Selector](/Content/images/doc/16_1/UiWidgets/Scheduler_ResourceHeaders.png)

Read the [Group Appointments by Resources](/Documentation/Guide/UI_Widgets/Scheduler/Resources/#Group_Appointments_by_Resources) topic to learn how to set grouping.
