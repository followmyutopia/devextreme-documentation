If an HTML element should display a DevExtreme icon, add a [dx-icon-IconName](/Documentation/ApiReference/UI_Widgets/CSS_Classes/#dx-icon-IconName) class to it. We recommend using inline HTML elements (`<i>` or `<span>`).

    <!--HTML-->
    <i class="dx-icon-email"></i>

You can find icon names in the [Built-In Icon Library](/Documentation/Guide/Themes_and_Styles/Icons/#Built-In_Icon_Library) article.
