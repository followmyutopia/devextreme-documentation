The **following desktop browsers** are supported in the **dx.mobile.js** library.

- ![Microsoft Edge](/Content/images/doc/16_1/PhoneJS/edge.png) Microsoft Edge
- ![Internet Explorer](/Content/images/doc/16_1/PhoneJS/ie.png) Internet Explorer 9+
- ![Safari](/Content/images/doc/16_1/PhoneJS/safari.png) Safari *Two Latest Versions* (note that Safari for Windows is not supported)
- ![Firefox](/Content/images/doc/16_1/PhoneJS/firefox.png) Firefox *Two Latest Versions*
- ![Google Chrome](/Content/images/doc/16_1/PhoneJS/chrome.png) Google Chrome *Two Latest Versions*
- ![Opera](/Content/images/doc/16_1/PhoneJS/opera.png) Opera *Two Latest Versions*

The **browsers integrated in the following mobile OS** are supported in the **dx.mobile.js** library.

- ![Android](/Content/images/doc/16_1/PhoneJS/android.png) Android 4+
- ![iOS Safari](/Content/images/doc/16_1/PhoneJS/safari-ios.png) iOS 6+
- ![IE Windows Phone](/Content/images/doc/16_1/PhoneJS/ie-windows-phone.png) Windows Phone 8+
