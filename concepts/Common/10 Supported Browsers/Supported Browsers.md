DevExtreme supports the following browsers.

**Desktop browsers**

- ![Microsoft Edge](/Content/images/doc/16_2/PhoneJS/edge.png) Microsoft Edge
- ![Internet Explorer](/Content/images/doc/16_2/PhoneJS/ie.png) Internet Explorer 9+
- ![Safari](/Content/images/doc/16_2/PhoneJS/safari.png) Safari *Two Latest Versions* (note that Safari for Windows is not supported)
- ![Firefox](/Content/images/doc/16_2/PhoneJS/firefox.png) Firefox *Two Latest Versions*
- ![Google Chrome](/Content/images/doc/16_2/PhoneJS/chrome.png) Google Chrome *Two Latest Versions*
- ![Opera](/Content/images/doc/16_2/PhoneJS/opera.png) Opera *Two Latest Versions*

**Browsers integrated in the following mobile OS**

- ![Android](/Content/images/doc/16_2/PhoneJS/android.png) Android 4+
- ![iOS Safari](/Content/images/doc/16_2/PhoneJS/safari-ios.png) iOS 6+
- ![IE Windows Phone](/Content/images/doc/16_2/PhoneJS/ie-windows-phone.png) Windows Phone 10
