DevExtreme supports the following browsers.

**Desktop browsers**

- ![Microsoft Edge](/Content/images/doc/19_2/PhoneJS/edge.png) Microsoft Edge
- ![Internet Explorer](/Content/images/doc/19_2/PhoneJS/ie.png) Internet Explorer 11
- ![Safari](/Content/images/doc/19_2/PhoneJS/safari.png) Safari - two latest versions (Safari for Windows is not supported)
- ![Firefox](/Content/images/doc/19_2/PhoneJS/firefox.png) Firefox - two latest versions
- ![Google Chrome](/Content/images/doc/19_2/PhoneJS/chrome.png) Google Chrome - two latest versions
- ![Opera](/Content/images/doc/19_2/PhoneJS/opera.png) Opera - two latest versions

**Browsers integrated in the following mobile OS**

- ![Android](/Content/images/doc/19_2/PhoneJS/android.png) Android 4.4+
- ![iOS Safari](/Content/images/doc/19_2/PhoneJS/safari-ios.png) iOS 9+

**Browsers integrated in the following frameworks**

- ![Electron](/Content/images/doc/19_2/PhoneJS/electron.png) Electron - the latest version
