You can set a custom palette for a chart, gauge or map using the <b>palette</b> configuration option. Assign an array of colors to this field.

    <!--JavaScript-->var chartOptions = {
        palette: ['#7CBAB4', '#92C7E2', '#75B5D6', '#B78C9B', '#F2CA84', '#A7CA74'],
        //...	
    };

[note] When you use this approach for a vector map, assign an array with only two colors to the **palette** option. See the [Reference](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#palette) for specific details on using palettes in **VectorMap**.
