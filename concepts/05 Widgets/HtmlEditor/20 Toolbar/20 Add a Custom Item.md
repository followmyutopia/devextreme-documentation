The toolbar can contain different DevExtreme widgets other than [buttons and select boxes](/Documentation/Guide/Widgets/HtmlEditor/Toolbar/Predefined_Items/). Assign the widget's name to the [widget](/Documentation/ApiReference/UI_Widgets/dxHtmlEditor/Configuration/toolbar/items/#widget) option and configure it in the [options](/Documentation/ApiReference/UI_Widgets/dxHtmlEditor/Configuration/toolbar/items/#options) object. In this case, you need to implement all the logic.

You can find an example in the [widget](/Documentation/ApiReference/UI_Widgets/dxHtmlEditor/Configuration/toolbar/items/#widget) description or the following demo:
