To configure map markers, use the [options](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/) of the *marker* layer.

To enable the user to distinguish one marker from another, accompany each marker with a label. For this purpose, set the [dataField](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/label/#dataField) option of the **label** object to a data field. This data field is taken from the [markers' data source](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#dataSource). Then, change the font size of the labels using the **size** option of the [font](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/label/font/) configuration object. In addition, change the color of a marker in the selected state using the [selectedColor](/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#selectedColor) option.
