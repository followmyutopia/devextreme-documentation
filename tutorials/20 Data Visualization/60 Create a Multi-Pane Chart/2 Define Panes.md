To define panes, assign an array of objects to the [panes](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/panes/) option. Each object configures one pane. Name the panes using the **name** option. From now on, you can refer to a particular pane by its name.

By default, all series are bound to the pane that is listed last in the **panes** array. Learn how to bind the series to different panes in the final step.
