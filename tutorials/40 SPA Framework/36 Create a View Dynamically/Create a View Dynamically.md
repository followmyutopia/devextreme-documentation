When an application is initialized, the HTML templates for views and layouts that are referenced on the application page are loaded. When displaying a view, an appropriate template is searched within the loaded templates. There can be scenarios when a view template cannot be defined beforehand, and a markup is generated dynamically or should be loaded from a file during the application flow. In this tutorial, you will learn how to load view templates dynamically so that they are found when displaying a view.
