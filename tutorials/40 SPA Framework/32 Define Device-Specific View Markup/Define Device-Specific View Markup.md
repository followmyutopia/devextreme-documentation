The DevExtreme framework is useful when you need to develop an application for different platforms. Just build a single application and pack it for different platforms. There can, however, be nuances that are particular to a specific device in some views. In this instance, you can implement device-specific views. In this tutorial, you will implement a simple application that will have the "home" view displayed differently in different platforms.
