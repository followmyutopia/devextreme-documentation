To try customizing the 'SlideOut' layout, use a [sample slide-out application](/Documentation/Tutorial/SPA_Framework/Use_a_SlideOut_Menu/#Apply_the_SlideOut_Layout). Open the **SlideOut**/*SlideOutLayout.html* file. In this file, you will find several layout markups - each for a certain platform. Add the [Toolbar](/Documentation/ApiReference/UI_Widgets/dxToolbar/) widget as the last element within the [dxTransition](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxTransition/) element of Android and Windows Phone layout markups. For iOS and Generic layouts, enclose the toolbar element to a [contentPlaceholder](/Documentation/ApiReference/SPA_Framework/Markup_Components/dxContentPlaceholder/) element. Turn the added toolbar into a command container by applying the **data-options** attribute set to **dxCommandContainer**. Specify a name for this command container using the **id** configuration option of the **dxCommandContainer** markup component.
