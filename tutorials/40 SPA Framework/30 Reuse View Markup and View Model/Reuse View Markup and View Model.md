In this tutorial, you will learn how to define a block of view markup elements separately and then use it in other views where this block is required. In addition, this block will be bound to the ViewModel of the view in which it is currently displayed. The set of ViewModel fields required for the reused markup will be factored out into a reusable ViewModel.
