Specify the columns that you need to display in the grid using the grid [columns'](/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/columns/) configuration option.

The **DataGrid** widget assumes [remote sorting](/Documentation/Guide/UI_Widgets/Data_Grid/Use_Custom_Store/#Remote_Operations/Basic_Remote_Operations) by default. Since the remote data service does not support sorting, in this tutorial, sorting will be performed locally. For this purpose, set the **sorting** field of the [remoteOperations](/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/remoteOperations/) configuration object to *false*.

For details on the **DataGrid** widget configuration options, refer to the [Configure DataGrid](/Documentation/Tutorial/UI_Widgets/Configure_DataGrid/#Configure_DataGrid) tutorial and the [Reference](/Documentation/ApiReference/UI_Widgets/dxDataGrid/) section.
