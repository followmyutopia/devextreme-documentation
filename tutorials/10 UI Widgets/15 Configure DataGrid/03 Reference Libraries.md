First, reference files required by the **DataGrid** on your HTML page. Note that jQuery and DevExtreme libraries are essential.

In addition to these scripts, **DataGrid** requires at least two stylesheet files. These mandatory files are the "dx.common.css", which provides the core styles and a theme-file that provides the distinctive look for **DataGrid**. You can find the libraries and stylesheets in your [DevExtreme package](/Documentation/Guide/Common/DevExtreme_Packages/).
