In this tutorial, you will learn how to configure the **PivotGrid** widget for displaying data from a remote OLAP server using the [jQuery](http://jquery.com/) approach. As an example, you will implement a pivot grid containing **Adventure Works** sample cube data. The pivot grid can be used to analyze this large data and understand it in general.
