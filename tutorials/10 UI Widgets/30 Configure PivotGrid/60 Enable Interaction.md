Finally, you can give an end-user more interaction capabilities by setting the following options to *true*.

* Options that affect the header's context menu.

	* [allowExpandAll](/Documentation/ApiReference/UI_Widgets/dxPivotGrid/Configuration/#allowExpandAll)			
		Allows an end-user to expand/collapse all header items within a header level.

	* [allowSorting](/Documentation/ApiReference/UI_Widgets/dxPivotGrid/Configuration/#allowSorting)			
		Allows an end-user to change sorting options.

	* [allowSortingBySummary](/Documentation/ApiReference/UI_Widgets/dxPivotGrid/Configuration/#allowSortingBySummary)			
		Allows an end-user to sort columns by summary values.

* Options that affects the Field Chooser.

	* [allowFiltering](/Documentation/ApiReference/UI_Widgets/dxPivotGrid/Configuration/#allowFiltering)			
		Allows an end-user to change filtering options.
