  According to the application map, there should be a 'Back' button that returns you to the previous view. You do not have to implement the 'Back' button. It is added automatically to the appropriate position on the current device (if there is no 'Back' button on the device). For details, refer to the [Navigate Back in Mobile Apps](/Documentation/Guide/SPA_Framework/Navigation_and_Routing/#Navigate_Back_in_Mobile_Apps) topic.
