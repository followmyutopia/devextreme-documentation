To clear search criteria and show data items in the initial order, pass **null** to the **sortData** function.

Add the "clear" button to *markup.html* and specify the **click** event handler for this button. Insert the **sortData** function call to the body of the **click** event handler. Pass **null** to the **sortData** function to clear sorting options.
