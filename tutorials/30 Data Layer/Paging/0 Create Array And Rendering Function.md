Begin by creating an array that will be accessed via the [DataSource](/Documentation/ApiReference/Data_Layer/DataSource/). Create the **EMPLOYEES** variable and assign the required array to it. You can find the **EMPLOYEES** array in the *employees.js* file.

For demonstration purposes, create the **render** function that displays data in the &lt;table&gt; element located on the *markup.html* page. You can see the render function implementation in the *renderer.js* file.

Initially, the table only displays column headers.
