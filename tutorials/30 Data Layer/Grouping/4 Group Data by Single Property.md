To group data items by a single property, the [group expression](/Documentation/Guide/Data_Layer/Data_Layer/#Reading_Data/Grouping) should be a string containing the required property name. Pass the "City" string to the **groupData** function to group data by the **City** property value.

Add the **button** element with the **id** attribute set to "button1" to *markup.html*. To apply grouping options described in this step each time you click this element, place the **groupData** function call to the **click** event handler of the added **button** element.
